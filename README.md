# Vue2-Online-Shop

## Tech Stack

- Vue.Js 2
- vue-router
- Pinia Store
- Axios
- Bootstrap 5
- Vuelidate
- Animate.css

## [Live Demo](https://youssefstore.netlify.app/)

## Usage

- cd vue2-online-shop
- npm install

- ### Dev server: run

 ``` shell
npm run dev
```
or
- ### Build for production: run

 ``` shell
npm run build
```
